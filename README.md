# Hyper TicTacToe

# Requirement  
    1. SFML-3.0.2 (Included in repository)
    2. CMake uses mingw and ninja to build (so either use your own building system or download mingw64 and ninja and keep them in toolchains folder)

# Features
    1. Simple but functional game (as of now not many features are available)
    2. Moves recommendations system (not yet implemented)
