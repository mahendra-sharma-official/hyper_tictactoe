# Hyper TicTacToe

# Installation
1. You can download the release version (Keep the res folder in the same place as application executable)

# Requirement  
    1. SFML-3.0.2 (Included in repository)
    2. CMake uses mingw and ninja to build (so either use your own building system or setup mingw and ninja for easy compilation)

# Features
    1. Simple but functional game (as of now not many features are available)
    2. Moves recommendations system (not yet implemented)
